# Portfólio Pessoal - Breno Abreu

Este repositório contém o código-fonte do meu portfólio pessoal, uma aplicação frontend criada para apresentar minha trajetória, projetos, habilidades, hobbies, princípios profissionais e canais de contato.

A aplicação foi desenvolvida como uma single page application, com conteúdo disponível em português e inglês, interface responsiva, microinterações e atenção a acessibilidade, performance e organização visual.

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS
- PrimeReact
- Lucide React
- React Icons

## Características

- Aplicação exclusivamente frontend, sem backend, API própria ou banco de dados.
- Estrutura organizada por seções e componentes reutilizáveis.
- Alternância entre português e inglês com persistência da preferência no navegador.
- Navegação por seções em página única.
- Galerias de fotos para hobbies.
- Conteúdo visual e textual voltado à apresentação profissional.

## Como executar localmente

Instale as dependências:

```bash
npm install
```

Execute o ambiente de desenvolvimento:

```bash
npm run dev
```

Gere a versão de produção:

```bash
npm run build
```

## Privacidade e dados

Este projeto não utiliza autenticação, backend, banco de dados, cookies de rastreamento ou coleta de dados pessoais dos visitantes.

As informações pessoais exibidas no site, como nome, email, links profissionais, currículo e localização geral, fazem parte do conteúdo público do portfólio.

Antes de reutilizar ou publicar alterações neste projeto, verifique se nenhum arquivo privado, credencial, chave de API, token, documento sensível ou dado pessoal não intencional foi incluído no repositório.
